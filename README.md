# Sudoku Solver

命令行数独求解器，使用 C 语言编写。

## 编译

```bash
make
```

## 使用方法

### 方式一：81位字符串（0 或 . 表示空）

```bash
./sudoku "530070000600195000098000060800060003400803001700020006060000280000419005000080079"
```

### 方式二：逐行输入

```bash
./sudoku
530070000
600195000
098000060
800060003
400803001
700020006
060000280
000419005
000080079
```
